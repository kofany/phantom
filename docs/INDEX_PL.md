## 🆘 Potrzebujesz pomocy?

### Kolejność działań przy problemie:

1. **Sprawdź FAQ** w [SZYBKI_START_PL.md](SZYBKI_START_PL.md) - Najczęstsze problemy
2. **Przeczytaj Troubleshooting** w [DOKUMENTACJA_PL.md](DOKUMENTACJA_PL.md)
3. **Włącz debug mode**: `./psotnic -d bot.cfg`
4. **Przeczytaj logi** - często zawierają rozwiązanie
5. **Sprawdź CHANGELOG** - Czy to znany problem/feature?

### Zgłaszanie błędów

Przygotuj:
- Wersję Psotnic (`./psotnic -v`)
- System operacyjny
- Pełny błąd (screenshot/tekst)
- Kroki do reprodukcji
- Logi (jeśli możliwe)
- Backtrace (jeśli crash)
